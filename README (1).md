## نشر الموقع على GitHub Pages

يمكنك نشر الموقع مجاناً بهذه الخطوات:

1. أنشئ ملف `.nojekyll` في جذر المشروع (لمنع GitHub من معالجة الملفات كـ Jekyll):
	```bash
	touch .nojekyll
	```
2. ادفع الملفات إلى فرع `gh-pages`:
	```bash
	git checkout --orphan gh-pages
	git add .
	git commit -m "نشر الموقع"
	git push origin gh-pages --force
	```
3. من إعدادات المستودع على GitHub، اختر `gh-pages` كفرع للنشر.
4. بعد دقائق سيكون الموقع متاحاً على:
	`https://<اسم-المستخدم>.github.io/<اسم-المستودع>/`

## معاينة محلية

```bash
python3 -m http.server 8000
# ثم افتح http://localhost:8000
```
